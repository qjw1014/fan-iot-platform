# 前端工程

本目录为工业风机物联网私有云平台的 Vue3 + TypeScript + Vite 前端工程。

## 已完成页面

- 登录页
- 首页仪表盘
- 客户管理
- 项目管理
- 盒子管理
- 设备管理
- 实时监控
- 历史数据
- 告警中心
- AI接口管理
- 用户管理
- 系统设置

## Phase 7 前端能力

- 盒子管理显示经度、纬度、地址、省市区、最后定位时间。
- 设备管理显示客户、项目、设备位置、地图坐标。
- 首页仪表盘增加设备地图模块，区分在线、离线、告警设备。
- AI接口管理支持实时数据、历史数据、JSON/CSV导出调试。
- 用户管理支持账号分页、新增、编辑、删除和角色分配。
- 系统设置显示系统操作日志。
- 所有新增页面和菜单使用简体中文。

## 本地启动

```powershell
npm install
npm run dev
```

默认地址：

```text
http://localhost:5173
```

## 编译

```powershell
npm run build
```
